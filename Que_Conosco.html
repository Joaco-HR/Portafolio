PROGRAMACIÓN CON BLOQUES

Aprendí programación con bloques en la primaria utilizando el programa Scratch. Durante mi tiempo usando Scratch, aprendí a diseñar tanto fondos como personajes, a utilizar correctamente los bloques de movimiento, apariencia, sonido, eventos y controles, y luego, en menor medida, los sensores, operadores y variables. Logré desarrollar diferentes programas y juegos que, desgraciadamente, perdí.

Nivel de conocimiento: 8/10 (primer lenguaje que aprendí)

Python

Aprendí a programar en Python en 2.º año del secundario hasta ahora. Lo primero que aprendí fue la sintaxis básica, como la definición y manejo de variables. Luego, el uso del print y los strings (diseño de mensajes). Después, aprendí a usar las funciones de decisión como if y los bucles for y while. Más adelante, aprendí la creación y manejo de listas, diccionarios y matrices. También aprendí a crear y codificar funciones y módulos (con def), y finalmente, la programación orientada a objetos (POO), definiendo clases y llamándolas desde otros programas. Además, aprendí a utilizar librerías externas.

También aprendí a utilizar a un alto nivel la librería pygame, con la cual desarrollé varios proyectos escolares. En menor medida, aprendí a usar las librerías random, json, tkinter, entre otras de distintos tipos.

Nivel de conocimiento: 9/10 (El lenguaje que mejor manejo y domino)

HTML y CSS

Aprendí a programar con HTML y CSS en 2.º año del secundario hasta ahora.

De HTML he aprendido a crear títulos, subtítulos, párrafos, cargar imágenes y ajustar sus tamaños. También aprendí a crear listas, vincular elementos de la misma página o de otras páginas HTML, así como a crear botones, implementar div y nav, y separar contenidos en diferentes secciones.

De CSS he aprendido a diseñar páginas HTML aplicando colores a fondos y textos, usando márgenes, bordes y rellenos para estructurar los elementos. Manejo flexbox para alinear contenido y uso propiedades como position: fixed y z-index para organizar elementos en capas. También sé aplicar sombras, transiciones y efectos al pasar el cursor para mejorar la interacción. He trabajado con distintas tipografías y estilos de texto, y sé dividir el diseño en secciones con clases específicas, dándoles un estilo propio y coherente.

En resumen, mis conocimientos de HTML y CSS me han permitido crear páginas visualmente agradables, interactivas y bien estructuradas, aplicando buenas prácticas de diseño y organización del código.

Nivel de conocimiento HTML: 8/10

Nivel de conocimiento CSS: 7/10

QBasic

Aprendí a programar en QBasic en 3.º año. Durante ese tiempo lo usé para traducir ejercicios hechos en Python, aprendiendo así a usar este lenguaje más antiguo. Aprendí a definir variables, imprimir elementos, cambiarles el color y la posición, usar estructuras if y case, bucles for y do. Más adelante, trabajé con listas y matrices, y lo último que vi fue la definición de funciones sencillas.

Nivel de conocimiento: 6/10

Arduino y C++

Aprendí a programar con Arduino desde 2.º año, aunque me enfoqué más seriamente en 3.º año. Durante este tiempo trabajé con Tinkercad y diferentes componentes como LEDs, pulsadores, potenciómetros, sensores de temperatura, servos, sensores de distancia, buzzers, sensores analógicos con ADC, motores y sensores tipo puntero H. Aprendí a estructurar correctamente los programas con void setup() y void loop(), a definir variables y a controlar la intensidad o el funcionamiento de los componentes. Además, utilicé condicionales if y bucles while. También tengo experiencia usando la librería de servos, el manejo de sensores analógicos y conozco el uso de interrupciones para responder a eventos en tiempo real.

Debido a que el lenguaje de Arduino está inspirado en C++, tuve un período donde aprendí a realizar algunas cosas aparte de lo mencionado para ayudar a mi hermano (que estudia electrónica) con algunos de sus proyectos. Allí aprendí a trabajar con una pantalla TFT LCD, aprendiendo en el proceso algunos elementos más avanzados de C++ y algunas librerías relacionadas con dicha pantalla.

Nivel de conocimiento Arduino: 8/10

Nivel de conocimiento C++: 7/10
